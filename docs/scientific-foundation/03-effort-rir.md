# Semana 4: Esfuerzo y RIR (Reps in Reserve)

## Resumen Ejecutivo

El RIR (Reps in Reserve) es la métrica que cuantifica qué tan cerca del fallo muscular se trabaja en cada serie. Este documento detalla la **gestión científica del esfuerzo** basada en 43 imágenes de evidencia del Lic. Jay Ehrenstein (APEKS/Icoan/NFA).

**Puntos Clave:**
- **Zona óptima**: RIR 0-3 (fallo o 1-3 reps antes)
- **Compuestos**: RIR 2-3 (evitar fallo técnico)
- **Aislados**: RIR 0-1 (maximizar estímulo)
- **Gestión de fatiga**: Intra-sesión e inter-sesión

---

## 1. Fundamentos de RIR (Imágenes 1-10)

### 1.1 Definición y Escala

**RIR (Reps in Reserve)**:
- Número de repeticiones que podrías hacer antes del fallo muscular concéntrico
- Escala: 0 (fallo) a 5+ (muy alejado del fallo)

**Escala RIR**:
```
RIR 0: Fallo muscular absoluto (no puedes hacer 1 rep más)
RIR 1: Podrías hacer 1 rep más y fallar
RIR 2: Podrías hacer 2 reps más
RIR 3: Podrías hacer 3 reps más
RIR 4: Podrías hacer 4 reps más
RIR 5+: Alejado del fallo (no efectivo para hipertrofia)
```

**Equivalencia con RPE**:
```
RIR 0 = RPE 10 (máximo esfuerzo)
RIR 1 = RPE 9
RIR 2 = RPE 8
RIR 3 = RPE 7
RIR 4 = RPE 6
```

**Evidencia Científica**:
- Helms et al. (2016): Validación escala RIR
- Zourdos et al. (2016): Precisión autorregulación
- Steele et al. (2021): RIR vs. fallo para hipertrofia

### 1.2 Zona Óptima de Estimulación (Imágenes 11-15)

**Rango Efectivo**: RIR 0-4
- **Máximo estímulo**: RIR 0-3
- **Estímulo moderado**: RIR 4
- **Estímulo mínimo**: RIR 5+

**Curva Estímulo-Fatiga**:
```
RIR 0: Estímulo 100%, Fatiga 100%
RIR 1: Estímulo 95%, Fatiga 70%
RIR 2: Estímulo 90%, Fatiga 50%
RIR 3: Estímulo 80%, Fatiga 35%
RIR 4: Estímulo 65%, Fatiga 25%
RIR 5: Estímulo 40%, Fatiga 15%
```

**Hallazgo Clave**: RIR 0-3 produce hipertrofia similar, pero RIR 2-3 tiene mejor ratio estímulo/fatiga.

**Evidencia Científica**:
- Grgic et al. (2022): Meta-análisis proximidad al fallo
- Nóbrega et al. (2018): Fallo no superior si volumen igualado
- Carroll et al. (2019): Fatiga acumulada RIR 0 vs. RIR 2

## 2. Asignación de RIR por Contexto (Imágenes 16-28)

### 2.1 Por Tipo de Ejercicio

#### Compuestos Multiarticulares
**RIR Recomendado**: 2-3

**Razones**:
- Riesgo de fallo técnico
- Alta fatiga central (SNC)
- Compromiso de seguridad
- Reclutamiento suficiente sin fallo

**Ejemplos**:
```
Sentadilla: RIR 2-3
Peso Muerto: RIR 3-4 (especialmente pesados)
Press Banca: RIR 2-3
Dominadas: RIR 1-2 (menor riesgo técnico)
```

#### Aislados Monoarticulares
**RIR Recomendado**: 0-1

**Razones**:
- Bajo riesgo técnico
- Fatiga localizada (no sistémica)
- Maximizar estímulo metabólico
- Seguros al fallo

**Ejemplos**:
```
Curl Bíceps: RIR 0-1
Extensiones Cuádriceps: RIR 0-1
Elevaciones Laterales: RIR 0-1
Curl Femoral: RIR 0-1
```

### 2.2 Por Zona de Intensidad (Imágenes 29-32)

#### Heavy (1-6 reps, 85-100% 1RM)
**RIR Recomendado**: 2-4

**Razones**:
- Preservar técnica bajo carga máxima
- Evitar lesiones
- Fatiga neural muy alta
- Reclutamiento ya máximo

**Ejemplo**:
```
Sentadilla 5 reps @ 85% 1RM
RIR objetivo: 2-3
```

#### Moderate (6-15 reps, 70-85% 1RM)
**RIR Recomendado**: 1-3

**Razones**:
- Balance estímulo/fatiga
- Zona más versátil
- Permite empujar según ejercicio

**Ejemplo**:
```
Press Inclinado 10 reps @ 75% 1RM
RIR objetivo: 1-2 (compuesto)

Cruces Mancuernas 12 reps @ 70% 1RM
RIR objetivo: 0-1 (aislado)
```

#### Light (16-30 reps, 60-70% 1RM)
**RIR Recomendado**: 0-1

**Razones**:
- Maximizar estrés metabólico
- Menor riesgo a altas reps
- Pump muscular
- Finishers

**Ejemplo**:
```
Peck Deck 20 reps @ 65% 1RM
RIR objetivo: 0 (fallo metabólico)
```

### 2.3 Por Posición en Sesión (Imágenes 33-37)

#### Primera Serie
**RIR Recomendado**: 3-4

**Razones**:
- Calentamiento específico
- Establecer groove motor
- No comprometer series subsecuentes

#### Series Intermedias (2-3)
**RIR Recomendado**: 2-3

**Razones**:
- Trabajo efectivo principal
- Estímulo óptimo sin fatiga excesiva
- Reserva para última serie

#### Última Serie
**RIR Recomendado**: 0-2

**Razones**:
- Maximizar estímulo
- No hay series posteriores
- All-out si aislado, conservador si compuesto

**Ejemplo Progresión Intra-Serie**:
```
Press Banca 4 sets x 8 reps
Set 1: RIR 3-4 (activación)
Set 2: RIR 2-3 (trabajo)
Set 3: RIR 2 (trabajo)
Set 4: RIR 1-2 (empuje final)
```

## 3. Gestión de Fatiga (Imágenes 38-43)

### 3.1 Fatiga Intra-Sesión

**Progresión RIR dentro de Sesión**:
```
Ejercicio 1 (primer compuesto): RIR 2-3
Ejercicio 2 (segundo compuesto): RIR 2-3
Ejercicio 3 (aislado 1): RIR 1-2
Ejercicio 4 (aislado 2): RIR 0-1
Ejercicio 5 (finisher): RIR 0
```

**Principio**: A medida que avanza la sesión, se puede/debe reducir RIR porque:
- Fatiga acumulada reduce output absoluto
- Ejercicios finales son aislados (seguros al fallo)
- Maximizar volumen efectivo

### 3.2 Fatiga Inter-Sesión

**RIR Semanal Progresivo**:
```
Semana 1: RIR promedio 3 (conservador, adaptación)
Semana 2: RIR promedio 2 (intensificación)
Semana 3: RIR promedio 1-2 (pico de esfuerzo)
Semana 4: RIR promedio 4-5 (deload)
```

**Monitoreo**:
- Si rendimiento cae (menos reps/peso) → aumentar RIR
- Si recuperación es pobre → aumentar RIR
- Si motivación baja → aumentar RIR temporalmente

### 3.3 Fatiga Sistémica vs. Local

**Alta Fatiga Sistémica** (deadlifts pesados, sentadillas):
- Aumentar RIR (3-4)
- Reducir frecuencia
- Descansos más largos

**Alta Fatiga Local** (aislados, pump work):
- Mantener RIR bajo (0-1)
- Aumentar frecuencia OK
- Descansos cortos

## 4. Implicaciones Prácticas

### 4.1 Para el Motor V3

**Effort Engine debe**:
1. Asignar RIR según categoría de ejercicio
2. Ajustar RIR según intensidad (heavy/moderate/light)
3. Progresar RIR dentro de sesión
4. Periodizar RIR entre semanas
5. Monitorear y adaptar según feedback

**Algoritmo de Asignación**:
```dart
class EffortEngine {
  int calculateRIR({
    required Exercise exercise,
    required IntensityZone zone,
    required int setNumber,
    required int totalSets,
    required int weekNumber,
    required int totalWeeks,
  }) {
    // Base RIR por tipo
    int baseRIR = switch (exercise.category) {
      'compound' when zone == IntensityZone.heavy => 3,
      'compound' when zone == IntensityZone.moderate => 2,
      'compound' when zone == IntensityZone.light => 2,
      'isolation' => 1,
      _ => 2,
    };
    
    // Ajuste por set number (reducir en últimas series)
    if (setNumber == totalSets && exercise.category == 'isolation') {
      baseRIR = max(0, baseRIR - 1); // All-out en última serie de aislados
    }
    
    // Ajuste por semana (intensificación)
    if (weekNumber == totalWeeks - 1) {
      baseRIR = min(5, baseRIR + 2); // Deload
    } else if (weekNumber >= totalWeeks * 0.66) {
      baseRIR = max(0, baseRIR - 1); // Intensificación final
    }
    
    return baseRIR.clamp(0, 5);
  }
  
  String getRIRGuidance(int rir) {
    return switch (rir) {
      0 => 'Fallo muscular. No puedes hacer 1 rep más.',
      1 => 'Podrías hacer 1 rep más, luego fallo.',
      2 => 'Podrías hacer 2 reps más. Esfuerzo muy alto.',
      3 => 'Podrías hacer 3 reps más. Esfuerzo alto.',
      4 => 'Podrías hacer 4 reps más. Esfuerzo moderado.',
      _ => 'Más de 4 reps en reserva. Poco estimulante.',
    };
  }
}
```

### 4.2 Para Workout Log Processor

**Detección de Desviaciones**:
```dart
class WorkoutLogProcessor {
  void analyzeEffortCompliance(WorkoutLog log) {
    for (var exerciseLog in log.exercises) {
      for (var setLog in exerciseLog.sets) {
        // Comparar RIR planificado vs. realizado
        final planned = getPrescription(exerciseLog.exerciseId).rir;
        final actual = setLog.actualRIR;
        
        if ((actual - planned).abs() > 2) {
          // Alerta: desviación significativa
          _flagRIRDeviation(
            exercise: exerciseLog.exerciseId,
            planned: planned,
            actual: actual,
            reason: _inferReason(actual, planned),
          );
        }
      }
    }
  }
  
  String _inferReason(int actual, int planned) {
    if (actual > planned + 2) {
      return 'Carga muy ligera o fatiga excesiva';
    } else if (actual < planned - 2) {
      return 'Sobreesfuerzo o sobreestimación de capacidad';
    }
    return 'Dentro de rango aceptable';
  }
}
```

### 4.3 Para Monitoring Engine

**Tracking de Esfuerzo Efectivo**:
```dart
class MonitoringEngine {
  int countEffectiveSets(WorkoutLog log) {
    int effectiveSets = 0;
    
    for (var exerciseLog in log.exercises) {
      for (var setLog in exerciseLog.sets) {
        // Series con RIR 0-4 cuentan como efectivas
        if (setLog.actualRIR <= 4) {
          effectiveSets++;
        }
      }
    }
    
    return effectiveSets;
  }
  
  int countHardSets(WorkoutLog log) {
    int hardSets = 0;
    
    for (var exerciseLog in log.exercises) {
      for (var setLog in exerciseLog.sets) {
        // Series con RIR 0-3 son "hard sets"
        if (setLog.actualRIR <= 3) {
          hardSets++;
        }
      }
    }
    
    return hardSets;
  }
  
  bool isOverreaching(List<WorkoutLog> weekLogs) {
    final totalHardSets = weekLogs
      .map((log) => countHardSets(log))
      .reduce((a, b) => a + b);
    
    // Alerta si >90% del volumen es hard sets
    final totalSets = weekLogs
      .map((log) => log.exercises.length)
      .reduce((a, b) => a + b);
    
    return (totalHardSets / totalSets) > 0.9;
  }
}
```

## 5. Ejemplos de Aplicación

### Ejemplo 1: Sesión de Pecho (Hipertrofia)
```
1. Press Banca 4x8
   Set 1: 8 reps @ RIR 3 (80kg)
   Set 2: 8 reps @ RIR 2 (80kg)
   Set 3: 8 reps @ RIR 2 (80kg)
   Set 4: 8 reps @ RIR 1 (80kg)

2. Press Inclinado Mancuernas 3x10
   Set 1: 10 reps @ RIR 2 (30kg c/u)
   Set 2: 10 reps @ RIR 2 (30kg c/u)
   Set 3: 10 reps @ RIR 1 (30kg c/u)

3. Cruces Cable 3x15
   Set 1: 15 reps @ RIR 1 (25kg)
   Set 2: 15 reps @ RIR 1 (25kg)
   Set 3: 15 reps @ RIR 0 (25kg, fallo)

4. Peck Deck 2x20 (finisher)
   Set 1: 20 reps @ RIR 0 (40kg, fallo)
   Set 2: 20 reps @ RIR 0 (40kg, fallo)
```

### Ejemplo 2: Progresión Semanal (Cuádriceps)
```
Semana 1 (Adaptación):
- Sentadilla 4x8 @ RIR 3
- Prensa 3x10 @ RIR 2
- Extensiones 3x15 @ RIR 1
Promedio RIR: 2.0

Semana 2 (Trabajo):
- Sentadilla 4x8 @ RIR 2
- Prensa 3x10 @ RIR 2
- Extensiones 3x15 @ RIR 0
Promedio RIR: 1.3

Semana 3 (Intensificación):
- Sentadilla 4x8 @ RIR 2
- Prensa 3x10 @ RIR 1
- Extensiones 3x15 @ RIR 0
Promedio RIR: 1.0

Semana 4 (Deload):
- Sentadilla 3x8 @ RIR 4
- Prensa 2x10 @ RIR 4
- Extensiones 2x15 @ RIR 3
Promedio RIR: 3.7
```

### Ejemplo 3: Ajuste por Fatiga
```
Sesión planificada:
- Peso Muerto 4x5 @ RIR 2

Registro real (fatiga alta):
- Set 1: 5 reps @ RIR 3 (más conservador)
- Set 2: 5 reps @ RIR 3
- Set 3: 4 reps @ RIR 3 (reducción de reps)
- Set 4: OMITIDO (fatiga excesiva)

Acción del motor:
- Detectar patrón de fatiga
- Reducir volumen próxima sesión (-1 set)
- Considerar deload anticipado
```

## 7. Términos Clave

- **RIR**: Reps in Reserve (repeticiones en reserva)
- **RPE**: Rating of Perceived Exertion (escala de esfuerzo percibido)
- **Fallo muscular**: Incapacidad de completar repetición adicional con técnica correcta
- **Fallo técnico**: Degradación de técnica antes de fallo muscular
- **Hard sets**: Series con RIR 0-3
- **Effective sets**: Series con RIR 0-4
- **EIMD**: Exercise-Induced Muscle Damage (daño muscular inducido por ejercicio)
