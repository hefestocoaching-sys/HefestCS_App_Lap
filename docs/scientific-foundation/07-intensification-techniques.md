# Técnicas de Intensificación Avanzadas

## Resumen Ejecutivo

Las técnicas de intensificación son métodos avanzados para aumentar la dificultad y estímulo del entrenamiento más allá del incremento convencional de volumen o carga. Este documento detalla las **3 técnicas principales validadas científicamente** basadas en 8 imágenes de evidencia del Lic. Jay Ehrenstein (APEKS/Icoan/NFA).

**Puntos Clave:**
- **Rest-Pause**: Series con micro-descansos para acumular volumen efectivo
- **Drop Sets**: Series descendentes para prolongar estimulación muscular
- **Bi-Series/Supersets**: Ejercicios pareados (compuestos o antagonistas)
- **Aplicación estratégica**: Solo cuando progresión convencional se agota
- **Usuarios avanzados**: Mayor beneficio en experiencia >2 años

---

## 1. Conceptos Fundamentales

### 1.1 Principios de Intensificación (Imágenes 1-2)

#### ¿Qué es la Intensificación?

Las técnicas de intensificación son **métodos para aumentar el estrés metabólico y mecánico** dentro de una sesión de entrenamiento sin incrementar linealmente el volumen total de series.

**Diferencia con progresión convencional**:
```
Progresión convencional:
- Añadir series: 3 sets → 4 sets
- Incrementar carga: 100kg → 105kg
- Aumentar frecuencia: 2x/semana → 3x/semana

Intensificación:
- Mismas series, más tensión por set (Rest-Pause)
- Mismo peso inicial, mayor volumen total (Drop Sets)
- Misma duración, mayor densidad (Supersets)
```

#### Cuándo Usar Técnicas de Intensificación

**✅ Indicaciones apropiadas**:
1. **Fase de intensificación** en mesociclo (semanas 5-6 antes de deload)
2. **Estancamiento de progresión** en ejercicio específico
3. **Limitación de tiempo** (mayor densidad de entrenamiento)
4. **Especialización** de músculo rezagado
5. **Usuarios avanzados** (>2 años experiencia)

**❌ Contraindicaciones**:
1. **Principiantes** (<1 año experiencia) - riesgo de sobrecarga
2. **Fase de acumulación** - interferencia con progresión volumen
3. **Déficit calórico severo** (>500 kcal) - recuperación comprometida
4. **Fatiga acumulada** - riesgo de sobreentrenamiento
5. **Ejercicios técnicamente complejos** (squat, deadlift pesados) - riesgo de lesión

#### Evidencia General

**Meta-Análisis**:
- **Schoenfeld et al. (2017)**: "Mechanisms of Hypertrophy"
  - Estrés metabólico (vía intensificación) contribuye a hipertrofia
  - Efecto complementario a tensión mecánica

- **Goto et al. (2005)**: "Effects of Resistance Exercise on Lipolysis"
  - Técnicas de intensificación aumentan respuesta hormonal aguda
  - Mayor acumulación de metabolitos (lactato, H+, fosfato inorgánico)

**Limitaciones**:
- Efectividad menor que progresión de volumen/carga en principiantes
- Mayor fatiga sistémica requiere gestión de recuperación
- No sustituyen fundamentos (volumen, intensidad, RIR)

### 1.2 Rest-Pause Sets (Imágenes 3-4)

#### Definición

**Rest-Pause** es una técnica donde se realiza una serie al fallo (o cerca), se descansa brevemente (10-30 segundos), y se continúa con repeticiones adicionales hasta acumular un volumen objetivo.

#### Protocolo Estándar

**Estructura**:
```
1. Serie inicial: Al fallo o 0 RIR (ej: 100kg x 8 reps)
2. Descanso: 15-20 segundos
3. Mini-serie 1: Máximo posible (ej: 100kg x 3 reps)
4. Descanso: 15-20 segundos
5. Mini-serie 2: Máximo posible (ej: 100kg x 2 reps)
6. (Opcional) Mini-serie 3: Máximo posible (ej: 100kg x 1 rep)

Total: 8+3+2+1 = 14 reps con 100kg
(vs. serie convencional: ~8 reps con 100kg)
```

#### Mecanismo de Acción

**Fisiológico**:
1. **Fosfocreatina (PCr)**: Recuperación parcial en 15-20 segundos (~50% restauración)
2. **Reclutamiento máximo**: Todas las mini-series cerca/al fallo = fibras tipo II activadas
3. **Tiempo bajo tensión**: Acumulación total mayor que serie convencional

**Metabólico**:
- Acumulación sostenida de metabolitos (lactato, H+)
- Mayor señalización anabólica (mTOR, MAPK)
- Estrés celular prolongado

#### Evidencia Científica

**Estudios Clave**:
- **Goto et al. (2004)**: "Muscular Adaptations to Combinations of High- and Low-Intensity Resistance Exercises"
  - Rest-Pause superior a series convencionales en hipertrofia
  - n = 26 sujetos entrenados, +8.2% vs. +5.1% grosor muscular

- **Prestes et al. (2019)**: "Understanding Rest-Pause Training"
  - Mayor respuesta hormonal (GH, testosterona) vs. series tradicionales
  - Efecto agudo, relevancia para hipertrofia crónica incierta

#### Aplicación Práctica

**Ejercicios recomendados**:
- ✅ Máquinas (Leg Press, Chest Press, Lat Pulldown)
- ✅ Aislamiento (Curls, Extensions, Raises)
- ⚠️ Compuestos con barra (Bench, Squat) - solo con spotter
- ❌ Ejercicios técnicos (Olympic lifts, Deadlift pesado)

**Frecuencia de uso**:
- **1-2 series Rest-Pause por músculo/sesión** (no todo el entrenamiento)
- **Último set de ejercicio** (acumulación final de fatiga)
- **Fase de intensificación** (semanas 5-6 de mesociclo)

**Ejemplo de integración**:
```
Bench Press:
- Set 1: 100kg x 8 @ 2 RIR
- Set 2: 100kg x 8 @ 2 RIR
- Set 3: 100kg x 8 @ 1 RIR
- Set 4 (Rest-Pause): 100kg x 8 + 15s + 3 + 15s + 2 = 13 reps totales
```

#### Consideraciones

**Ventajas**:
- ✅ Mayor volumen con misma carga
- ✅ Alta eficiencia de tiempo
- ✅ Reclutamiento completo de fibras

**Desventajas**:
- ❌ Fatiga sistémica elevada
- ❌ Requiere alta tolerancia al dolor/incomodidad
- ❌ Riesgo de fallo técnico en ejercicios complejos

### 1.3 Drop Sets (Imágenes 5-6)

#### Definición

**Drop Sets** (series descendentes) consisten en realizar una serie al fallo, reducir inmediatamente la carga 20-30%, y continuar al fallo nuevamente. Proceso puede repetirse 2-3 veces.

#### Protocolo Estándar

**Estructura (Double Drop)**:
```
1. Serie inicial: Al fallo con carga objetivo (ej: 100kg x 8 reps)
2. Reducción inmediata: -25% carga (75kg)
3. Continuación: Al fallo (ej: 75kg x 10 reps)
4. (Opcional) Reducción adicional: -25% nuevamente (55kg)
5. (Opcional) Continuación final: Al fallo (ej: 55kg x 12 reps)

Total: 8+10+12 = 30 reps en ~60 segundos
```

**Variantes**:
- **Mechanical Drop Set**: Cambiar ángulo/grip en lugar de carga
  - Ejemplo: Pull-ups (pronado, fallo) → Chin-ups (supinado, fallo)
  
- **Strip Sets**: Reducción rápida de carga (barras con discos)
  - Ejemplo: Deadlift 140kg (fallo) → quitar 2x10kg → 120kg (fallo)

#### Mecanismo de Acción

**Fisiológico**:
1. **Fatiga progresiva**: Reducción de carga compensa fatiga acumulada
2. **Reclutamiento completo**: Todas las series al fallo = activación máxima
3. **Tiempo bajo tensión extendido**: 60-90 segundos continuos

**Metabólico**:
- **Máxima acumulación metabólica**: Lactato, H+, fosfato inorgánico
- **Hinchazón celular** (cell swelling): Señal anabólica potente
- **Respuesta hormonal aguda**: GH y otras hormonas anabólicas

#### Evidencia Científica

**Estudios Clave**:
- **Fink et al. (2018)**: "Effects of Drop Set Resistance Training on Acute Indicators of Hypertrophy"
  - Drop Sets: Mayor tiempo bajo tensión y volumen total vs. series tradicionales
  - Similar activación muscular (EMG)

- **Varović et al. (2021)**: "Drop Set vs. Traditional Resistance Training"
  - Hipertrofia similar entre Drop Sets y series tradicionales cuando volumen igualado
  - Drop Sets ventajosos para eficiencia de tiempo

- **Schoenfeld & Grgic (2018)**: "Can Drop Set Training Enhance Muscle Growth?"
  - Revisión: Evidencia preliminar favorable, necesita más investigación
  - Posible herramienta para superar mesetas

#### Aplicación Práctica

**Ejercicios recomendados**:
- ✅✅✅ Máquinas (ajuste rápido de carga): Leg Press, Cable Exercises
- ✅✅ Mancuernas (drop rápido): DB Curls, Lateral Raises
- ✅ Barras con discos (strip sets): Bench Press, Deadlift (con asistente)
- ❌ Ejercicios técnicamente demandantes sin asistencia

**Frecuencia de uso**:
- **1-2 series Drop Set por músculo/sesión**
- **Último set de último ejercicio** para ese músculo
- **No más de 2-3 veces/semana** (alta fatiga)

**Ejemplo de integración**:
```
Entrenamiento de hombros (día push):
1. Overhead Press: 4x6 @ 2 RIR
2. Lateral Raise: 3x12 @ 2 RIR
3. Cable Lateral Raise (Drop Set final):
   - 10kg x 12 (fallo)
   - Inmediato: 7.5kg x 15 (fallo)
   - Inmediato: 5kg x 20 (fallo)
   Total: 47 reps en 70 segundos
```

#### Consideraciones

**Ventajas**:
- ✅ Máxima eficiencia de tiempo
- ✅ Alto estímulo metabólico (hinchazón celular)
- ✅ "Pump" muscular intenso

**Desventajas**:
- ❌ Fatiga extrema (dificulta ejercicios posteriores)
- ❌ Recuperación prolongada (24-48h+)
- ❌ Requiere equipo de rápido ajuste

### 1.4 Bi-Series y Supersets (Imágenes 7-8)

#### Definición

**Bi-Series** o **Supersets** son pares de ejercicios realizados consecutivamente sin descanso (o descanso mínimo <30 segundos) entre ellos.

#### Tipos de Supersets

**1. Supersets Antagonistas** (recomendado):
- **Estructura**: Músculo agonista → Músculo antagonista
- **Ejemplos**:
  - Bench Press (pecho) → Barbell Row (espalda)
  - Leg Extension (cuádriceps) → Leg Curl (isquios)
  - Bicep Curl → Tricep Extension

**Mecanismo**:
- **Inhibición recíproca**: Activación de antagonista facilita recuperación del agonista
- **Eficiencia**: 2 ejercicios en tiempo de 1.5 ejercicios
- **Perfusión sanguínea**: Flujo alternado mejora eliminación de metabolitos

**2. Supersets Compuestas** (mismo músculo):
- **Estructura**: Ejercicio compuesto → Aislamiento del mismo músculo
- **Ejemplos**:
  - Squat → Leg Extension (cuádriceps)
  - Pull-up → Bicep Curl (espalda/bíceps)
  - Bench Press → Cable Fly (pectorales)

**Mecanismo**:
- **Pre-fatiga** o **post-fatiga**: Aislamiento asegura fatiga completa del músculo objetivo
- **Volumen concentrado**: Máximo estímulo en tiempo mínimo

**3. Supersets No Competitivas**:
- **Estructura**: Músculos sin interferencia (ej: tren superior + tren inferior)
- **Ejemplos**:
  - Bench Press → Calf Raise
  - Bicep Curl → Plank

**Mecanismo**:
- **Eficiencia pura**: Reducción de tiempo de entrenamiento
- **Sin sinergia fisiológica especial**

#### Evidencia Científica

**Estudios Clave**:

**Supersets Antagonistas**:
- **Robbins et al. (2010)**: "The Effect of an Upper Body Agonist-Antagonist Resistance Training Protocol"
  - Supersets antagonistas: Mismo volumen en 50% menos tiempo
  - Sin pérdida de rendimiento vs. series tradicionales

- **Ciccone et al. (2014)**: "Paired Set Training: Muscular and Metabolic Responses"
  - Mayor gasto energético y respuesta metabólica
  - Volumen total preservado

**Supersets Compuestas**:
- **Fisher et al. (2014)**: "Evidence-Based Resistance Training Recommendations"
  - Pre-fatiga no superior a orden tradicional para hipertrofia
  - Útil para "sentir" músculo objetivo (conexión mente-músculo)

- **Brentano et al. (2017)**: "Effects of Different Exercise Order Strategies"
  - Compuesto → Aislamiento = mejor rendimiento que inverso
  - Preferir post-fatiga (aislamiento después de compuesto)

#### Aplicación Práctica

**Supersets Antagonistas** (recomendado para hipertrofia):
```
Ejemplo Upper/Pull session:
1A. Barbell Row: 4x8 @ 2 RIR
1B. Bench Press: 4x8 @ 2 RIR
(Alternar: Row → descanso 30s → Bench → descanso 90s → repetir)

2A. Lat Pulldown: 3x10 @ 2 RIR
2B. Overhead Press: 3x10 @ 2 RIR

3A. Bicep Curl: 3x12 @ 2 RIR
3B. Tricep Extension: 3x12 @ 2 RIR
```

**Supersets Compuestas** (para especialización):
```
Ejemplo especialización de cuádriceps:
1. Squat: 4x8 @ 2 RIR (descanso 2-3 min)
2A. Leg Press: 3x10 @ 2 RIR
2B. Leg Extension: 3x15 @ 1 RIR (inmediato tras Leg Press)
```

**Frecuencia de uso**:
- **Supersets antagonistas**: 2-3 pares/sesión (muy tolerable)
- **Supersets compuestas**: 1-2 pares/sesión (más fatigante)
- **No toda la sesión**: Mantener ejercicios principales (squat, deadlift) sin superseries

#### Consideraciones

**Ventajas**:
- ✅ Alta eficiencia de tiempo (30-40% reducción)
- ✅ Supersets antagonistas: Mantienen rendimiento
- ✅ Mayor densidad de entrenamiento

**Desventajas**:
- ❌ Requiere disponibilidad de equipo (gimnasio lleno = difícil)
- ❌ Supersets compuestas: Mayor fatiga local
- ❌ Demanda cardiovascular elevada (puede limitar rendimiento)

## 2. Implementación en Motor V3

### 2.1 Sistema de Selección de Técnicas

Motor V3 recomienda técnicas de intensificación estratégicamente:

```dart
/// lib/features/training/domain/engines/intensification_techniques_engine.dart

enum IntensificationTechnique {
  restPause,       // Rest-Pause sets
  dropSet,         // Drop sets
  antagonistSuperset,  // Supersets antagonistas
  compoundSuperset,    // Supersets compuestas
  none,            // Sin técnica
}

class IntensificationTechniquesEngine {
  /// Determina técnica apropiada para ejercicio/contexto
  static IntensificationTechnique selectTechnique({
    required Exercise exercise,
    required TrainingPhase phase,
    required ExperienceLevel userLevel,
    required bool timeConstrained,
    required int weekInMesocycle,
  }) {
    // Regla 1: Solo usuarios intermedios/avanzados
    if (userLevel == ExperienceLevel.beginner) {
      return IntensificationTechnique.none;
    }

    // Regla 2: Solo en fase de intensificación O limitación de tiempo
    final isIntensificationPhase = phase == TrainingPhase.intensification;
    if (!isIntensificationPhase && !timeConstrained) {
      return IntensificationTechnique.none;
    }

    // Regla 3: Ejercicios técnicos nunca usan intensificación extrema
    if (_isTechnicalExercise(exercise)) {
      // Solo supersets antagonistas permitidos (sin compromiso técnico)
      return timeConstrained
          ? IntensificationTechnique.antagonistSuperset
          : IntensificationTechnique.none;
    }

    // Regla 4: Priorizar por contexto
    if (timeConstrained) {
      // Máxima eficiencia = supersets antagonistas
      return IntensificationTechnique.antagonistSuperset;
    }

    if (isIntensificationPhase) {
      // Fase intensificación = Rest-Pause o Drop Sets
      return _selectIntensiveTechnique(exercise);
    }

    return IntensificationTechnique.none;
  }

  /// Identifica ejercicios técnicamente complejos
  static bool _isTechnicalExercise(Exercise exercise) {
    return exercise.movementPattern == MovementPattern.olympic ||
        exercise.movementPattern == MovementPattern.squat ||
        exercise.movementPattern == MovementPattern.hinge;
  }

  /// Selecciona técnica intensiva (Rest-Pause vs. Drop Set)
  static IntensificationTechnique _selectIntensiveTechnique(
    Exercise exercise,
  ) {
    // Drop Sets requieren ajuste rápido de carga
    final isQuickAdjustable = exercise.equipment == Equipment.machine ||
        exercise.equipment == Equipment.cable ||
        exercise.equipment == Equipment.dumbbells;

    if (isQuickAdjustable) {
      return IntensificationTechnique.dropSet;
    }

    // Rest-Pause para barras o ejercicios sin ajuste rápido
    return IntensificationTechnique.restPause;
  }

  /// Aplicar técnica a set
  static IntensifiedSet applyTechnique({
    required IntensificationTechnique technique,
    required ExerciseSet baseSet,
  }) {
    switch (technique) {
      case IntensificationTechnique.restPause:
        return _applyRestPause(baseSet);
      case IntensificationTechnique.dropSet:
        return _applyDropSet(baseSet);
      case IntensificationTechnique.none:
      case IntensificationTechnique.antagonistSuperset:
      case IntensificationTechnique.compoundSuperset:
        return IntensifiedSet.fromBase(baseSet); // Sin modificación
    }
  }

  /// Rest-Pause: Serie inicial + 2 mini-series
  static IntensifiedSet _applyRestPause(ExerciseSet baseSet) {
    return IntensifiedSet(
      weight: baseSet.weight,
      targetReps: baseSet.targetReps,
      targetRIR: 0, // Al fallo
      technique: IntensificationTechnique.restPause,
      restPauseProtocol: RestPauseProtocol(
        initialReps: baseSet.targetReps,
        restDuration: Duration(seconds: 15),
        miniSets: 2, // 2 mini-series adicionales
      ),
    );
  }

  /// Drop Set: Serie inicial + 2 drops
  static IntensifiedSet _applyDropSet(ExerciseSet baseSet) {
    return IntensifiedSet(
      weight: baseSet.weight,
      targetReps: baseSet.targetReps,
      targetRIR: 0, // Al fallo
      technique: IntensificationTechnique.dropSet,
      dropSetProtocol: DropSetProtocol(
        initialWeight: baseSet.weight,
        drops: [
          DropStage(
            weightReduction: 0.25, // -25%
            targetReps: null, // Al fallo
          ),
          DropStage(
            weightReduction: 0.25, // -25% adicional
            targetReps: null, // Al fallo
          ),
        ],
      ),
    );
  }
}

/// Set con técnica de intensificación aplicada
class IntensifiedSet {
  final double weight;
  final int targetReps;
  final double targetRIR;
  final IntensificationTechnique technique;
  final RestPauseProtocol? restPauseProtocol;
  final DropSetProtocol? dropSetProtocol;

  const IntensifiedSet({
    required this.weight,
    required this.targetReps,
    required this.targetRIR,
    required this.technique,
    this.restPauseProtocol,
    this.dropSetProtocol,
  });

  factory IntensifiedSet.fromBase(ExerciseSet baseSet) {
    return IntensifiedSet(
      weight: baseSet.weight,
      targetReps: baseSet.targetReps,
      targetRIR: baseSet.targetRIR,
      technique: IntensificationTechnique.none,
    );
  }
}

/// Protocolo Rest-Pause
class RestPauseProtocol {
  final int initialReps;
  final Duration restDuration;
  final int miniSets;

  const RestPauseProtocol({
    required this.initialReps,
    required this.restDuration,
    required this.miniSets,
  });
}

/// Protocolo Drop Set
class DropSetProtocol {
  final double initialWeight;
  final List<DropStage> drops;

  const DropSetProtocol({
    required this.initialWeight,
    required this.drops,
  });

  /// Calcula peso para cada drop stage
  List<double> get dropWeights {
    double currentWeight = initialWeight;
    return drops.map((drop) {
      currentWeight *= (1 - drop.weightReduction);
      return currentWeight;
    }).toList();
  }
}

class DropStage {
  final double weightReduction; // Porcentaje (ej: 0.25 = -25%)
  final int? targetReps;        // null = al fallo

  const DropStage({
    required this.weightReduction,
    this.targetReps,
  });
}

enum Equipment {
  barbell,
  dumbbells,
  machine,
  cable,
  bodyweight,
}
```

### 2.2 Sistema de Supersets

Motor V3 agrupa ejercicios en supersets cuando apropiado:

```dart
/// lib/features/training/domain/engines/superset_pairing_engine.dart

class SupersetPairingEngine {
  /// Crear pares de supersets antagonistas
  static List<ExercisePair> createAntagonistSupersets({
    required List<Exercise> exercises,
  }) {
    final pairs = <ExercisePair>[];
    final used = <String>{};

    for (final exercise in exercises) {
      if (used.contains(exercise.id)) continue;

      // Buscar antagonista
      final antagonist = _findAntagonist(exercise, exercises);
      if (antagonist != null && !used.contains(antagonist.id)) {
        pairs.add(ExercisePair(
          exerciseA: exercise,
          exerciseB: antagonist,
          type: SupersetType.antagonist,
          restBetweenExercises: Duration(seconds: 30),
          restBetweenSupersets: Duration(seconds: 90),
        ));
        used.add(exercise.id);
        used.add(antagonist.id);
      }
    }

    return pairs;
  }

  /// Buscar músculo antagonista
  static Exercise? _findAntagonist(
    Exercise exercise,
    List<Exercise> available,
  ) {
    final primaryMuscle = exercise.primaryMuscles.first;
    final antagonistMuscle = _getAntagonistMuscle(primaryMuscle);

    if (antagonistMuscle == null) return null;

    // Buscar ejercicio que trabaje músculo antagonista
    return available.firstWhereOrNull(
      (e) => e.primaryMuscles.contains(antagonistMuscle) &&
             e.id != exercise.id,
    );
  }

  /// Mapeo de músculos antagonistas
  static MuscleGroup? _getAntagonistMuscle(MuscleGroup muscle) {
    const antagonistMap = {
      MuscleGroup.pectorals: MuscleGroup.lats,
      MuscleGroup.lats: MuscleGroup.pectorals,
      MuscleGroup.biceps: MuscleGroup.triceps,
      MuscleGroup.triceps: MuscleGroup.biceps,
      MuscleGroup.quadriceps: MuscleGroup.hamstrings,
      MuscleGroup.hamstrings: MuscleGroup.quadriceps,
      MuscleGroup.deltoids: MuscleGroup.lats, // Press vs. Pull
    };

    return antagonistMap[muscle];
  }

  /// Crear superset compuesta (compuesto + aislamiento)
  static ExercisePair? createCompoundSuperset({
    required Exercise compound,
    required List<Exercise> isolationOptions,
  }) {
    // Buscar aislamiento del mismo músculo que el compuesto
    final primaryMuscle = compound.primaryMuscles.first;
    final isolation = isolationOptions.firstWhereOrNull(
      (e) => e.primaryMuscles.contains(primaryMuscle) &&
             e.movementPattern == MovementPattern.isolation,
    );

    if (isolation == null) return null;

    return ExercisePair(
      exerciseA: compound,
      exerciseB: isolation,
      type: SupersetType.compound,
      restBetweenExercises: Duration(seconds: 0), // Inmediato
      restBetweenSupersets: Duration(seconds: 120),
    );
  }
}

class ExercisePair {
  final Exercise exerciseA;
  final Exercise exerciseB;
  final SupersetType type;
  final Duration restBetweenExercises;
  final Duration restBetweenSupersets;

  const ExercisePair({
    required this.exerciseA,
    required this.exerciseB,
    required this.type,
    required this.restBetweenExercises,
    required this.restBetweenSupersets,
  });
}

enum SupersetType {
  antagonist,  // Push + Pull
  compound,    // Compuesto + Aislamiento
  nonCompetitive, // Sin relación muscular
}
```

## 4. Implicaciones Prácticas

### 4.1 Guías de Aplicación

**Frecuencia de Intensificación por Nivel**:

| Nivel | Rest-Pause | Drop Sets | Supersets | Total Sets/Semana |
|-------|-----------|-----------|-----------|-------------------|
| **Principiante** | ❌ No | ❌ No | ✅ Supersets antagonistas solo | 0-2 |
| **Intermedio** | ⚠️ 1-2x/semana | ⚠️ 1-2x/semana | ✅ 3-4x/semana | 3-6 |
| **Avanzado** | ✅ 2-3x/semana | ✅ 2-3x/semana | ✅ 4-6x/semana | 6-10 |

**Fase de Mesociclo**:

| Fase | Técnicas Recomendadas | Razón |
|------|----------------------|-------|
| **Acumulación** | Supersets antagonistas | Eficiencia sin comprometer volumen |
| **Intensificación** | Rest-Pause, Drop Sets | Máximo estímulo antes de deload |
| **Deload** | Ninguna | Recuperación completa |

### 4.2 Errores Comunes

**❌ Error 1**: Usar técnicas en todos los sets
- **Correcto**: Solo último set de 1-2 ejercicios por músculo

**❌ Error 2**: Aplicar a ejercicios técnicos (squat, deadlift pesados)
- **Correcto**: Solo máquinas, cables, aislamiento

**❌ Error 3**: Combinar múltiples técnicas en mismo ejercicio
- **Correcto**: Una técnica por ejercicio (ej: no Rest-Pause + Drop Set)

**❌ Error 4**: Ignorar recuperación
- **Correcto**: Monitorear fatiga, reducir si rendimiento cae

### 4.3 Casos de Uso Motor V3

**Caso 1: Usuario avanzado en fase intensificación**

## 5. Términos Clave

| Término | Definición |
|---------|------------|
| **Intensificación** | Métodos para aumentar dificultad sin incrementar volumen/carga linealmente |
| **Rest-Pause** | Serie al fallo + micro-descansos + repeticiones adicionales |
| **Drop Set** | Serie al fallo + reducción inmediata de carga + continuar al fallo |
| **Superset** | Dos ejercicios consecutivos sin descanso (o mínimo) |
| **Superset antagonista** | Par de músculos opuestos (push/pull, flexores/extensores) |
| **Superset compuesta** | Compuesto + aislamiento del mismo músculo |
| **Pre-fatiga** | Aislamiento antes de compuesto |
| **Post-fatiga** | Aislamiento después de compuesto (recomendado) |
| **Estrés metabólico** | Acumulación de metabolitos (lactato, H+, fosfato inorgánico) |
| **Hinchazón celular** | Cell swelling - expansión de células musculares por metabolitos |
| **Inhibición recíproca** | Activación de antagonista facilita relajación del agonista |
| **Fosfocreatina (PCr)** | Sistema energético de corta duración (~10s), recupera 50% en 15-20s |
| **Tiempo bajo tensión** | Duración total de contracción muscular |
| **Mechanical Drop Set** | Cambio de ángulo/grip en lugar de carga |
| **Strip Set** | Drop set con barras (quitar discos rápidamente) |
